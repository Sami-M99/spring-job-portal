# 🌱 Section 01 - Spring Core & Maven Foundation

## 🎯 Overview

This section introduces the fundamental concepts behind the Spring Framework and Maven. It covers the problems of traditional Java development, the motivation for using Spring, the role of Dependency Injection (DI) and Inversion of Control (IoC), and the basics of Maven project management.

## 📚 Topics Covered

### ☕ Java vs Spring

* Challenges of traditional Java development
* Tight Coupling and Boilerplate Code
* Benefits of using Spring Framework

### 🌱 Spring Framework Fundamentals

* Spring Ecosystem Overview
* Spring Core Concepts
* Inversion of Control (IoC)
* Dependency Injection (DI)
* Loose Coupling

### 🚀 Spring Boot Introduction

* Auto Configuration
* Embedded Servers
* Dependency Management
* Faster Application Development

### 📦 Maven Essentials

* What is Maven?
* Dependency Management
* Maven Repositories
* Project Object Model (`pom.xml`)
* Maven Build Lifecycle
* Common Maven Commands

### 🧩 Spring Beans & IoC Container

* Bean Creation and Management
* `@Configuration`
* `@Bean`
* Application Context
* Bean Retrieval by Type and Name

## 🛠️ Technologies Used

* Java
* Spring Framework
* Maven
* IntelliJ IDEA

## 🚀 Learning Outcomes

After completing this section, you will be able to:

* Understand why Spring is widely used in modern Java applications.
* Explain IoC and Dependency Injection concepts.
* Create and register Spring Beans manually.
* Work with the Spring IoC Container.
* Manage project dependencies using Maven.
* Understand Maven project structure and lifecycle.
